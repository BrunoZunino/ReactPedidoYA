INTRODUCCIÓN

Este es el paso a paso para poder ejecutar la aplicación web de Picante la cuál está hecha en ReactJS.

PASO A PASO

Primero debemos ir a la consola y ejecutar el siguiente comando: npm install
Con este comando instalaremos todos los paquetes necesarios que se están usando en el archivo JSON de la aplicación.

En caso de querer solo descargar un paquete en particular, debemos hacer: npm install <nombre del paquete>

Una vez que tenemos las dependencias instaladas, abrimos la consola de nuestra máquina o del editor que estemos usando
y ejecutamos el siguiente comando para correr nuestro servidor NODE: node server.js

Ahora es momento de correr la aplicación de ReactJS en nuestro localhost. Para eso en otra consola realizamos
el siguiente comando: npm start.

IMPORTANTE

Debemos contar con NODEJS en nuestra máquina para poder ejecutar el comando "npm". Otra alternativa es usar YARN.

Al momento de correr ReactJS nos llevará al localhost 3000 en el browser, en el cuál es donde está alojada la aplicación.
